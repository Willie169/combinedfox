## combinedfox

`user.js` combining [arkenfox](https://github.com/arkenfox/user.js), [Betterfox](https://github.com/yokoffing/Betterfox), and combinedfox overrides for Firefox-based browsers on deskop, and instructions and extensions settings for Firefox-based browsers on desktop and Android, for privacy, security, and convenience.

Do not trust any site that claim to be combinedfox. This repo is the only official source.

The combinedfox user.js combines Arkenfox, Betterfox, and combinedfox overrides and is designed for Firefox-based browsers on deskop. Read the [wiki](https://github.com/Willie169/combinedfox/wiki) for how to use combinedfox `user.js` and instructions and extensions settings about Firefox-based browsers on desktop and Android.
